# User Flow for Demo Video

## Video Structure (3 minutes total)

### [0:00-0:20] The Problem (20 seconds)

**Visual:** Split screen — left side shows a busy sports bar with fans cheering, right side shows the same bar empty because they didn't know about the match.

**Narration:**
> "Running a local business during big events is chaos. You don't know which events are coming, how to promote, or how many customers to expect. Result? Empty bar during the World Cup final, or packed house with no beer left."

**Text overlay:** "Local businesses lose $X billion annually from missed event opportunities."

---

### [0:20-0:50] Meet EventBoost (30 seconds)

**Visual:** Business owner opens the EventBoost web app. Dashboard shows upcoming events, recommended actions, and ROI from past events.

**Narration:**
> "Meet EventBoost — the AI-powered platform that turns any event into your biggest sales day. No manual research. No guesswork. Just results."

**Demo steps:**
1. Open web app → Dashboard loads
2. Show upcoming events: "Argentina vs Brazil — Tomorrow 8pm"
3. Show agent recommendation: "High demand expected. Create promo?"
4. Click "Create Promo" button

**Text overlay:** "EventBoost — Your business + any event = more sales"

---

### [0:50-1:30] The Agent in Action (40 seconds)

**Visual:** Agent generates promotion, QR code appears, business owner approves.

**Narration:**
> "EventBoost's AI agent analyzes the event, predicts demand, and creates a complete promotion strategy — all in seconds."

**Demo steps:**
1. Agent generates promo: "Watch Argentina vs Brazil LIVE! Scan for 20% off wings + free first beer"
2. QR code appears instantly
3. Business owner customizes: changes discount to 25%, adds "Free nachos with any beer"
4. Agent recommends inventory: "Order 200 more beer bottles, 15 kg more wings"
5. Agent recommends staff: "Add 2 bartenders for 6-11pm shift"
6. Agent creates social media post: "Watch the World Cup here! Scan for deals 🍺⚽"
7. One-click approval → Everything is set

**Text overlay:** "One click. Promotion, inventory, staff, and marketing — all handled."

---

### [1:30-2:10] Customer Experience (40 seconds)

**Visual:** Customer scans QR code, lands on mobile page, signs up, pre-orders combo.

**Narration:**
> "Customers scan the QR code, see the deal, and pre-order their combo. No waiting in line. No surprise prices."

**Demo steps:**
1. Customer scans QR code → Mobile landing page loads
2. Shows event: "Argentina vs Brazil — Tomorrow 8pm"
3. Shows deal: "25% off wings + Free first beer + Free nachos"
4. Customer enters name, phone number
5. Selects combo: "Wings + Beer Combo — $15 (was $22)"
6. Mock payment → Confirmation screen
7. Gets reminder: "Match starts in 2 hours. See you at [Bar Name]!"

**Text overlay:** "Scan. Order. Show up. Simple."

---

### [2:10-2:40] Self-Improvement (Arize Phoenix) (30 seconds)

**Visual:** Open Phoenix dashboard, show traces from past events, demonstrate learning.

**Narration:**
> "EventBoost doesn't just run — it learns. Every event, every prediction, every outcome is traced and analyzed. The agent gets smarter over time."

**Demo steps:**
1. Open Phoenix dashboard
2. Show traces: "Predicted 150 customers for Brazil vs Serbia"
3. Show actual: "Actual: 130 customers"
4. Show adjustment: "Agent learned: Brazil group games = 1.2x multiplier"
5. Show next prediction: "Argentina vs Brazil knockout = 1.5x multiplier"
6. Show improvement graph: "Prediction accuracy improved from 70% to 92% over 10 events"

**Text overlay:** "AI that learns from every event. Gets better every time."

---

### [2:40-3:00] Results + Call to Action (20 seconds)

**Visual:** Show ROI dashboard, compare before/after EventBoost.

**Narration:**
> "Result? 30% more revenue per event. Zero stockouts. Perfect staffing. EventBoost — turn events into revenue."

**Demo steps:**
1. Show ROI dashboard: "Argentina vs Brazil — $2,400 extra revenue"
2. Show comparison: "Before EventBoost: $800/event. After: $1,040/event"
3. Show GitHub repo link
4. Show demo link

**Text overlay:**
- "EventBoost — Turn events into revenue."
- "github.com/mikaelaldy/event-boost"
- "Try the demo: [demo-link]"

---

## Key Moments to Highlight

1. **Agent takes action** — Not just suggestions, but actual promotions, QR codes, inventory orders
2. **One-click workflow** — Business owner approves, agent executes
3. **Customer experience** — Scan QR, pre-order, show up
4. **Self-improvement** — Agent learns from every event
5. **Measurable ROI** — Clear before/after comparison

---

## Video Tips

- **Keep it fast-paced** — 3 minutes goes by quickly
- **Show real UI** — Not mockups, but actual running app
- **Use text overlays** — Reinforce key points visually
- **Show the agent's reasoning** — Phoenix traces visible
- **End with impact** — ROI numbers, before/after comparison

---

## Recording Checklist

- [ ] Web app running locally or on Cloud Run
- [ ] Mock data seeded (events, businesses, customers)
- [ ] Phoenix dashboard open with traces
- [ ] QR code generator working
- [ ] Customer landing page mobile-friendly
- [ ] Agent chat responding correctly
- [ ] ROI dashboard showing numbers

- [ ] Script printed or on second screen
- [ ] Quiet environment for narration
